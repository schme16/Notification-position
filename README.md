# Notification Position Plus GNOME Shell Extension
This extension allows you to set the shell notifications position to any of your screen corners, as well as middle top and bottom, as well as what screen they will appear on.



## Installation

1. **Visit the GNOME Extensions Website**:
   - Go to the [GNOME Extensions page for Notification Position Plus](https://extensions.gnome.org/extension/8894/notification-position/).
